# What's Possible
Even though your bots not unlocked, there are still *lots* of things you can do with him! There are lots of things you can do like:

 - Use the Python SDK
    - Control and drive him around
    - Make him say things or throw his cube around
 - Setup Wire-Pod
    - Setup Custom Voice Commands in it
 - Install and try out older firmwares
    - Some favourites of mine are:
        - 1.6 (Last anki firmware)
        - 1.4 (Snow globe!)
 - Use Alexa
